Ubermelon Order Form
====================

Simple Flask app to demonstrate web forms.

To run locally::

  $ virtualenv-3.4 venv
  $ . venv/bin/activate
  (venv) $ pip install -r requirements.txt
  $ python mirror.py

This is already packed up to deploy at Heroku::

  $ heroku login
  $ heroku create
  $ git push heroku
